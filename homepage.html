This is simple crud api create in express js.

EndPoints

URL: /person

Method: GET

Description: Gives lists of person

URL: /person/add

Method: POST

Data: {"name": "Name Of person", "address": "Address of person"}

Description: >Add person, input data need to be in json format

URL: /person/edit/:id

Method: POST

Data: {"name": "Edited Name Of person", "address": "Edited Address of person"}

Description: >Edit the person, input data need to be in json format

URL: /person/delete/:id

Method: GET

Description: Deletes the user pass id in url
